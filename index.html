O desafio de hoje consiste em fazer esse CRUD funcionar corretamente. Os dados serão salvos, obtidos, alterados e removidos a partir do array DB.

Para executar as ações serão utilizados os métodos HTTP: GET, POST, PUT e DELETE

Também, para desafio serão utilizados conceitos do javascript como Array.find, Array.splice e Array.push.

Recomendo usar o Postman para fazer a chamada dos endpoints. Endpoints com métodos diferente de GET não são facilmente executados a partir do navegador

É essencial que existam 5 rotas:

  • GET /pessoa: Obtém a lista completa de pessoas.
  • GET /pessoa/:id: Obtém uma pessoa pelo ID ou retorna um erro 404 caso nao encontre.
  • POST /pessoa: Envia os dados da pessoa a serem salvos e retorna a pessoa já cadastrada no DB.
  • PUT /pessoa/:id: Envia o ID pela rota e os dados da pessoa a serem atualizados a partir do BODY e retorna a pessoa atualizada.
  • DELETE /pessoa/:id: Envia o ID pela rota, exclui essa pessoa e retorna a pessoa excluída.
